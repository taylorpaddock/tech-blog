# tech-blog

## Description

A tech blog site where developers can publish their blog posts and comment on other developersâ€™ posts as well. This app follows the MVC paradigm in its architectural structure, using Handlebars.js as the templating language, Sequelize as the ORM, and the express-session npm package for authentication.


## Installation

To install please clone this repository from GitHub on your terminal.

## Usage

Navigate to the root folder, run the following commands from the CLI:

Install packages
`npm i`

Seed DB
`npm run seed`

Start Server
`npm start server`

deployed applicaiton: https://tech-blog-uw.herokuapp.com/
